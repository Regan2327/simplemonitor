# SimpleMonitor - Blockchain Monitoring Tool

**Project Type:** Decentralized Application (DApp) on Flare Network  
**Contract Address:** `0xA311256948A1f7C5b92D63e642Ecd4b8D04816C6`

---

## Project Description

SimpleMonitor is a decentralized monitoring tool built on the Flare (Coston2) network.
It allows users to monitor endpoints, track status updates, and manage monitoring 
tasks using blockchain technology for transparency and security.

The application provides a user-friendly interface to interact with smart contracts,
demonstrating the integration of Web3 technologies with modern frontend frameworks.

---

## Features

✅ **Wallet Connection** - Connect MetaMask wallet seamlessly  
✅ **Smart Contract Interaction** - Direct blockchain communication  
✅ **Real-time Monitoring** - Track endpoint status updates  
✅ **Transaction Tracking** - View pending and confirmed transactions  
✅ **Responsive Design** - Works on desktop and mobile  
✅ **Error Handling** - Clean UI error messages  
✅ **Modular Architecture** - Well-organized code structure

## Technologies Used

### Frontend
- **Next.js 14** - React framework with server-side rendering
- **TypeScript** - Type-safe JavaScript
- **React** - UI component library
- **TailwindCSS** - Utility-first CSS framework

### Blockchain
- **Flare Network (Coston2)** - EVM-compatible blockchain
- **Solidity 0.8.0** - Smart contract programming
- **Wagmi** - React hooks for Ethereum
- **Viem** - TypeScript interface for Ethereum
